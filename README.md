# ربات کد

یک ربات تلگرام ساده با Cloudflare Worker که پیام‌های کاربران را دریافت می‌کند و واکنش 👍 می‌فرستد، و پیام تبلیغاتی با لینک گروه و اطلاعات تماس تنها در **چت خصوصی (پی وی)** کاربر ارسال می‌کند.

---

## ویژگی‌ها

- ارسال خودکار واکنش 👍 به هر پیام دریافتی.
- ارسال پیام تبلیغاتی با دکمه شیشه‌ای (Inline Keyboard) فقط در دایرکت کاربران.
- جلوگیری از ارسال پیام تبلیغاتی برای ربات‌ها یا خود ربات.
- طراحی شده برای اجرا روی Cloudflare Workers با امنیت بالا (استفاده از Environment Variables).

---

## نصب و راه‌اندازی

1. **ایجاد ربات تلگرام** و دریافت `BOT_TOKEN` و `BOT_USERNAME`.

2. **تنظیم Environment Variables در Cloudflare Worker:**

```
BOT_TOKEN=توکن-ربات
BOT_USERNAME=نام-کاربری-ربات
```

3. **آپلود کد به Cloudflare Worker** و تنظیم مسیر `/webhook` برای دریافت پیام‌ها.

4. **ست کردن وب‌هوک تلگرام:**

```
https://api.telegram.org/bot<BOT_TOKEN>/setWebhook?url=https://your-domain.com/webhook
```

> جایگزین کنید:
> - `<BOT_TOKEN>` با توکن ربات خودتان
> - `https://your-domain.com/webhook` با مسیر Cloudflare Worker شما

5. **تست ربات**: هر پیام جدید به ربات واکنش 👍 می‌گیرد و اگر پیام از کاربر خصوصی باشد، پیام تبلیغاتی با لینک گروه دریافت می‌کند.

## نکات مهم

- دکمه شیشه‌ای لینک گروه فقط در پی وی کاربر نمایش داده می‌شود.
- ربات پیام تبلیغاتی را به ربات‌ها یا خودش ارسال نمی‌کند.
- می‌توانید متن پیام تبلیغاتی را بر اساس نیاز خود تغییر دهید.
